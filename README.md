Este repositorio contiene mis implementaciones y experimentos en C++20, con un enfoque especializado en la programación de sistemas y la eficiencia en la ejecución de procesos. 
Mi objetivo principal es dominar el uso de procesadores multi-núcleo para exprimir al máximo la capacidad de cómputo paralela del hardware actual.

¿En qué me enfoco? Me centro en la gestión avanzada de hilos y el modelo de memoria de C++. Utilizo herramientas esenciales de sincronización como Mutex, Locks y Atomics para garantizar que el flujo de datos sea seguro y eficiente. Mi aprendizaje abarca desde el diseño de estructuras de datos lock-based hasta implementaciones lock-free, buscando siempre optimizar la latencia y el rendimiento.

Todo este trabajo sirve como complemento fundamental para mi estudio de RTOS (Real-Time Operating Systems) en el ámbito de los sistemas embebidos, donde la gestión precisa de tareas y el tiempo real son críticos.

Gestión del proyecto: Para mantener un estándar profesional y organizado, utilizo CMake como sistema de construcción. Esto me permite gestionar de forma automatizada los flags de compilación necesarios para hilos y corrutinas, asegurando que cada módulo del proyecto sea portable, fácil de escalar y esté correctamente vinculado a las librerías del sistema.
